# Page Types

- Cover
- Agenda
- Section Divider
- Problem Page
- Solution Page
- Comparison Page
- Process Page
- Timeline Page
- Cards Page
- Chart Page
- Case Page
- Conclusion Page

## Selection Rules

- Use cards for parallel points.
- Use process pages when order matters.
- Use comparison when a choice must be obvious.
- Use chart pages only when the numeric view changes the decision.
